# Nodemon

1. install nodemon

```console
 > npm i nodemon
```

    2. start server

```console
 > npx nodemon index.js
```

    or

    Setting in`./package.json`

```json
    "scripts": {
        ...
        "start": "nodemon ./bin/www"
	...
    },
```

    Start server

```console
 > npm start
```
